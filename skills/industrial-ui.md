# Industrial UI Skill

Design for industrial B2B business:
- Conveyor belts
- Bearings
- V-belts
- Timing belts
- Industrial chains

Style:
- Modern
- Clean
- Technical
- Trustworthy
- Professional

Colors:
- Navy
- White
- Gray
- Orange accent

UI rules:
- Responsive mobile first
- Clear spacing
- Strong CTA buttons
- Product cards must be readable
- Avoid overly decorative design
- Use icons where helpful
- Keep layout easy for factory/procurement customers
